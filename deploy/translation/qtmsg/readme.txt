README

Con l'introduzioni dei messaggi in spagnolo-argentina si e' reso necessario
rinominare il file qt_ar.ts (che per Qt riporta i messaggi in Arabo) in
qt_am.ts e fare una copia di qt_es.ts (spagnolo) chiamandolo qt_ar.ts.

Bisogna ricordarsi di fare queste operazioni ogni qual volta si aggiornano
i messaggi di sistema di Qt.
